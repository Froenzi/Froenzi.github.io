Dissertation & Software Artefact

My final-year university dissertation studied the viability of software as a method to educate the average person on the "Induced Traffic Demand" (ITD) phenomenon and how it impacts traffic congestion, through utilising interactivity and gamification.

The software artefact, linked in the repository below, walks users through an interactable traffic simulation that allows them to create and alter road layouts. Users can then simulate the outcome of their changes, comparing their impact to convey ITD.

The software served as a testing environment for the purposes of my dissertation, which is also available for download below.

Flip n' Flop

Direct3D12 (DirectX12) Game Project

Flip n' Flop is an orthographic 3D puzzle-platformer programmed in C++ using Direct3D12 as its graphics API.
The game engine was implemented from scratch using DirectX12 rendering samples as a base; possessing data-driven asset loading systems to faciliate level-editing in external software by non-programmer team members. (We opted to use Tiled Level Editor.)

The game's main gameplay feature is the ability to "flip" between "the back" and "the front" of the current level to bypass obstacles and avoid hazards not visible from the player's current perspective.

Wild Trance

Unity Engine Game Project

Wild Trance is a platformer game made in the Unity Engine; the player must make their way through a colourful forest while combatting corrupted forest animals.
The game equips the player character with a sword and a shield with a reflective parry mechanic, granting the ability to reflect projectiles back at enemy creatures.

This project was my first serious foray into developing games with the Unity Engine, as well as utilising Unity's organisation features for project management.
